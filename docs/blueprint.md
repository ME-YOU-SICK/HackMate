# **App Name**: HackMate

## Core Features:

- User Authentication: Secure sign-up, login, and role-based access for both participants and organizers.
- Profile Management: Rich user profiles including name, photo, age, city, tech stack, skills, past hackathons, projects, and linked social profiles (GitHub, LinkedIn).
- Event Participation via Unique Codes: Join hackathons instantly with 6-character event codes provided by organizers.
- Event Management: Organizers can create and manage hackathons, track participants, and post updates.
- Participant Discovery: View and browse other participants in the same event, with filters for skills, location, and experience.
- Team Matchmaking: AI-powered teammate suggestion tool based on skills, experience, and tech stack diversity.
- Connections & Networking: Follow, connect, and view mutual event history with other users.
- Messaging System: Direct messaging and group chat for team members and event participants.
- Personalized Dashboard: Role-specific dashboards with event updates, connection requests, and notifications.

## Style Guidelines:

- A luminous blend of Electric Blue (#29ABE2) and Royal Purple (#6A5ACD), creating a futuristic, tech-inspired vibe.
- Soft frosted-glass panels over a subtle deep navy-to-indigo gradient, providing a modern, airy depth.
- Vibrant Coral-Orange (#FF8C00) for calls-to-action and interactive highlights.
- 'Inter' for clean body text, paired with 'Poppins' or 'Sora' for headings to add a touch of futuristic elegance.
- Thin-line, minimal icons with subtle gradients or glassy effects for cohesion with the interface.
- Grid-based, responsive design with generous spacing and layered elements that create depth.
- Gentle fade, scale, and parallax effects on hover and scroll to create a sense of magic without overwhelming performance.
