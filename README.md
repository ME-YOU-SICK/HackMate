## ✨ Key Features

HackMate is designed to solve the biggest challenge in any hackathon: building a great team.

-   **🤖 AI-Powered Team Matching (`Teamify`):** Describe your ideal team and project, and let our AI find the best-fit members from your network.
-   **👤 Dynamic User Profiles:** Showcase your skills, past projects, and hackathon achievements to attract the right collaborators.
-   **🗓️ Event Discovery:** Browse upcoming hackathons and join with a simple event code.
-   **🤝 Seamless Networking:** Connect with other participants, organizers, and mentors before, during, and after the event.
-   **🛠️ Tools for Organizers:** Streamline event management and foster better participant engagement.
-   **🏆 Track Achievements:** Keep a record of your hackathon journey and celebrate your wins.

## 💻 Tech Stack

This project is built with a modern, type-safe, and scalable technology stack.

-   **Framework:** [Next.js](https://nextjs.org/) (App Router)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **AI/Generative:** [Google's Genkit](https://firebase.google.com/docs/genkit)
-   **UI:** [React](https://reactjs.org/)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **Component Library:** [Shadcn/ui](https://ui.shadcn.com/)
-   **Icons:** [Lucide React](https://lucide.dev/)

## 🏁 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   [Node.js](https://nodejs.org/en/) (v18.x or later recommended)
-   `npm` or `yarn` or `pnpm`

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/ME-YOU-SICK/hackmate.git
    cd hackmate
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```


3.  **Run the development server:**
    ```sh
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result!
